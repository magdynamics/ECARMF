# ECARMF-005 Glossary

## Evidence

A governed information object that supports an assertion, calculation, state, risk score, decision, forecast, AI output, or report.

## Context Package

A reproducible package of facts, assumptions, state, evidence, time, relationships, and conditions under which analysis or reasoning occurs.

## Traceability

The ability to connect an output to its requirements, source data, entities, evidence, context, services, APIs, tests, and approvals.

## Auditability

The ability to inspect, verify, and reconstruct platform behavior.

## Explainability

The ability to explain the basis of a recommendation, decision, score, threshold warning, or AI output.
