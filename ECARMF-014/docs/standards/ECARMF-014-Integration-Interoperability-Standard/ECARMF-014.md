# ECARMF-014 — Integration and Interoperability Standard

## Engineering Construction Specification

### 1. Standard Purpose

This standard establishes the mandatory construction requirements for Integration and Interoperability Standard.

### 2. Scope

This standard applies to ECARMF-compliant software platforms, services, APIs, databases, knowledge graphs, digital twins, AI agents, validation engines, and certification tooling implementing Integration and Interoperability Standard capabilities.
