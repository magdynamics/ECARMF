# AI Construction Guide

AI coding agents shall generate ECARMF code in this sequence:

1. Read approved ECARMF standards.
2. Build the ontology context.
3. Generate entities.
4. Generate persistence objects.
5. Generate repositories.
6. Generate domain services.
7. Generate APIs.
8. Generate events.
9. Generate Knowledge Graph mappings.
10. Generate Digital Twin mappings.
11. Generate tests.
12. Generate documentation.

AI shall preserve requirement traceability and shall not alter normative behavior.
