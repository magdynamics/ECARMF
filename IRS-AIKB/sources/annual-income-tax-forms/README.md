## Scope boundary

This first layer contains the principal return and its general instructions. It does not yet include every supporting schedule. Principal schedules—including Schedules C, E, F, SE, K-1, K-2, K-3, M-1, M-2, M-3, and the Form 990 schedule family—will be maintained as a linked schedule layer so form editions are not confused with schedule editions.
