# ECARMF Framework Overview

ECARMF connects asset facts with risk decisions.

The framework follows this sequence:

1. Define the asset.
2. Define the economic assumptions.
3. Define the capital structure.
4. Identify risk categories.
5. Assign risk indicators.
6. Establish thresholds.
7. Score risk.
8. Trigger decisions.
9. Monitor changes.
10. Report results.

## Primary Output

The primary ECARMF output is a decision-ready risk profile that explains whether an asset is within acceptable limits, approaching a warning zone, or exceeding action thresholds.
