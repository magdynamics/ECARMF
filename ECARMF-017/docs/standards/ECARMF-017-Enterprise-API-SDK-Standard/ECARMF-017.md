# ECARMF-017 — Enterprise API and SDK Standard

## Engineering Construction Specification

### 1. Standard Purpose

This standard establishes the mandatory construction requirements for Enterprise API and SDK Standard.

### 2. Scope

This standard applies to ECARMF-compliant software platforms, services, APIs, databases, knowledge graphs, digital twins, AI agents, validation engines, and certification tooling implementing Enterprise API and SDK Standard capabilities.
