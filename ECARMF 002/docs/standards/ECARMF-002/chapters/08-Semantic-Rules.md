# Chapter 8 — Semantic Rules

## Requirement SEM-0001
Every implementation shall implement ECARMF semantic rules.

## Identity Semantics
Identity is immutable. Changing an Entity Identifier creates a new Entity.

## Relationship Semantics
Relationships possess meaning, direction, context, temporal validity, evidence, confidence, and lifecycle.

## Composition Semantics
Composition implies lifecycle dependency.

## Aggregation Semantics
Aggregation defines grouping without ownership dependency.

## Temporal Semantics
Historical state shall never be destroyed.

## AI Semantics
Machine learning may augment but shall not redefine ontology.
