## Purpose

ECARMF-004 defines the engineering requirements for building Asset Intelligence within the ECARMF platform.

It translates the ECARMF principles, ontology, and reference architecture into a construction specification for:

- Economic Digital Assets (EDA)
- Economic Digital Cells (EDC)
- Asset Digital Twins
- Asset state models
- Asset intelligence pipelines
- Asset performance monitoring
- Asset risk and capital context
- AI-assisted asset reasoning
